# 俄罗斯方块 (Tetris)

使用 Python + Tkinter 实现的经典俄罗斯方块游戏。

## 运行

```bash
python3 tetris.py
```

无需安装任何依赖，使用 Python 自带的 Tkinter 库。

## 操作

| 按键 | 功能 |
|------|------|
| ← → | 左右移动 |
| ↑ | 旋转方块 |
| ↓ | 加速下落 |
| 空格 | 硬降（直接落底） |
| P | 暂停/继续 |
| R | 重新开始 |

## 功能

- 7 种经典方块（I、O、T、S、Z、J、L）
- 下一个方块预览
- 计分系统（消行越多分数越高）
- 等级系统（每消 10 行升一级，速度加快）
